# Theme-Windows

这是一个旨在让 Windows 个性化变得更方便，并让你的个性化方案更容易与他人分享的项目。

对于我们很多人来说，在电脑上设置一张好看的壁纸是很简单的，但有些人也可能希望把整个系统的提示音都换成自己喜欢的声优声音，或者希望每次打开电脑时都会听到一句“Ciallo”。

这些事情现在其实都可以做到，但你可能会不想经历那种繁琐的过程：去各个网站收集精美 CG，或者不断请求各种 AI 去模仿你喜欢的声优的声音（我以前就经常这么做）。之后再去到处找教程，比如“如何修改系统提示音”，也同样很麻烦。

Theme-Windows 的目标是让这些事情变得简单：只需要把资源放进去，或者直接使用 Discord 社区成员甚至未来官方提供的资源包即可。我们希望有一天，你的 Windows 可以像《赛博朋克 2077》或《星光咖啡馆与死神之蝶》那样具有完整的氛围感，只需下载一个由资深用户或朋友制作的 “.thwin” 文件即可实现。

## 使用方法

预期的使用方式是：

首先你会得到一个 “xxx.thwin” 文件，它可以来自朋友分享，甚至来自官方开发；或者你也可以准备一些图片、音频等资源。

然后使用 “Pack.exe” 来决定你想要的 Windows 个性化内容。两种方式最终都会生成一个 “xxx.thwin” 包，其中包含你对 Windows 个性化的全部设定信息。

最后只需要用 “Theme-Windows.exe” 打开该包，就可以让你的 Windows 按照预期方式完成个性化。
